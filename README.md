# Welcome to API with FastAPI!
Hi! To run the project.  

**First:**
Create a virtual enviroment with python =3.7
How to create a virtual enviroment [here](https://www.innovasoftware.dev/blog/post/entornos-virtuales-con-anaconda-miniconda/).

Clone the project with git clone url.  

**Second:**
With the virtual enviroment actived install the requirements, with pip install -r requirements.txt

 **Finally:**  
 
Start server
>uvicorn main:app --reload

Create user with username, password and active
Login with your credentials

>http://127.0.0.1:8000/docs
